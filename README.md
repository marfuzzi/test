<a name="headers"><h2>Заголовки</h2></a>












### Production
```
npm i
npm start
```
### Development
```
npm i
npm run dev
```
Открываем по адресу http://localhost:3000

**Команды**
* npm run lint запускает eslint и csscomb
* npm run test запускает mocha тесты
* npm run hermione запускает hermione тесты
