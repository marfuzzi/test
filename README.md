# Визуализация зрения робота.

Домашнее задание по теме "Мультимедиа".

Открываем по адресу https://marfuzzi.github.io/ya_multimedia/index.html

**Запуск проекта**
```
npm i
gulp
```
должен запуститися по адресу  http://localhost:3000

**Технологии**
* Препроцессор **sass**
* Сборка **gulp**
